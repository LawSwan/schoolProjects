## About Flowgorithm

Flowgorithm is a free application that allows you to create programs using flowcharts. The approach focuses on:

- **Visual Logic**: Design algorithms graphically before coding
- **Language Agnostic**: Concepts translate to any programming language
- **Step-by-Step Execution**: Watch your algorithm run in real-time
- **Export Options**: Convert flowcharts to actual code (Python, Java, C++, etc.)

## Viewing the Flowcharts

### Option 1: Flowgorithm Application
1. Download Flowgorithm from [flowgorithm.org](http://flowgorithm.org)
2. Open any `.fprg` file
3. Run the flowchart step-by-step

### Option 2: Export to Code
Flowgorithm can export flowcharts to:
- Python
- Java
- C++
- C#
- JavaScript
- And more...

### Option 3: Print/Export as Image
1. Open flowchart in Flowgorithm
2. File > Print or File > Export
3. Save as PNG/PDF for documentation
